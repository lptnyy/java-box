# Java-Box（盒子）
    docker是一个大鲸鱼，大鲸鱼上面拖着很多集装箱，而我想在集装箱里装满我的盒子，可以单部署，也可以集群部署
    让所有应用服务支持热部署，支持动态加载卸载模块,服务访问监控,支持多种访问方式 http https tcp rpc， 
    通过box后台让开发者专注模块开发，对外提供的服务接口可管理可监控，可自动化部署。
